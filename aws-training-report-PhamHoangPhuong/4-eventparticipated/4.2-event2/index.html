Event 2

Reflection Report: “FCAJ Community Day - Conference Call” – 23/05/2026

Purpose of the Event

The second FCAJ Community Day focused on the latest trends in Cloud Computing and Artificial Intelligence, advanced Prompt Engineering techniques, and in-depth knowledge of Amazon CloudFront. The event also introduced practical AI applications built on AWS and shared valuable career insights for students and young professionals interested in cloud technologies.

Speakers

  • Mr. Hưng
  • Mr. Thịnh
  • Guest Speakers

Key Highlights

Cloud and AI Market Trends

Mr. Hưng shared valuable insights into the current technology market and discussed the rapid growth of Cloud Computing and Artificial Intelligence. He explained how technological advancements are changing recruitment requirements and emphasized the importance of continuously improving both technical and soft skills.

Topics included:

  • Current recruitment trends in the Cloud Computing industry.
  • The increasing impact of Artificial Intelligence across various industries.
  • The technical and professional skills currently required by employers.
  • Strategies for building personal value and maintaining long-term career competitiveness.

Advanced Prompt Engineering

Mr. Thịnh introduced several advanced Prompt Engineering techniques that can significantly improve the quality and accuracy of responses generated by Large Language Models (LLMs). The session focused on writing clear prompts with sufficient context and selecting appropriate prompting strategies for different scenarios.

Topics Covered

  • Writing prompts with complete and meaningful context.
  • Role-based prompting techniques.
  • Step-by-step prompting strategies.
  • Using structured output formats to improve AI responses.
  • Common mistakes when interacting with AI systems and how to avoid them.
  • Introducing Obsidian as a "Second AI Brain" for effective personal knowledge management.

Friendly AI Assistant Demonstration

The speakers demonstrated an AI-powered assistant built using AWS services. The assistant was capable of answering user questions intelligently while integrating multiple cloud services to provide a practical enterprise AI solution.

  • Overview of the AI Assistant architecture.
  • Integration of AWS services for intelligent question answering.
  • Real-time demonstration of user interactions with the AI assistant.

Amazon CloudFront

One of the most technical sessions of the event focused on Amazon CloudFront. The speakers explained how CloudFront improves website performance, strengthens security, and reduces infrastructure costs by efficiently distributing content worldwide.

Key topics included:

  • The latest Amazon CloudFront pricing model.
  • Multi-layer caching architecture.
  • Origin Shield.
  • Origin Cloaking.
  • Signed URLs.
  • Viewer Access Control.
  • HTTP/3 support.
  • Performance optimization techniques.
  • AWS security best practices for content delivery.

Through practical demonstrations, I gained a much better understanding of how Amazon CloudFront accelerates web applications while simultaneously improving security and reducing operational costs.

Knowledge and Skills Gained

Prompt Engineering

  • Learned how to write more professional and structured prompts.
  • Understood the importance of providing sufficient context to AI models.
  • Improved my ability to generate more accurate and relevant AI responses.
  • Recognized the importance of selecting appropriate prompting techniques for different use cases.

Cloud Computing

  • Developed a deeper understanding of Amazon CloudFront architecture.
  • Learned how CDN technologies improve website performance.
  • Understood modern content delivery and caching strategies.
  • Learned practical security features available in CloudFront deployments.

Career Development

  • Gained a clearer understanding of the skills currently required by technology companies.
  • Learned strategies for building long-term technical expertise.
  • Recognized the importance of continuous learning in rapidly evolving technology fields.

Applying the Knowledge

After attending the event, I immediately incorporated several of the shared techniques into my daily learning activities.

  • Started using Obsidian to organize technical notes and learning materials.
  • Designed prompts with better structure and clearer context when using ChatGPT and Amazon Q.
  • Applied Amazon CloudFront concepts while completing AWS laboratory exercises.
  • Continued studying CDN technologies and AWS security services through official documentation.

These practical changes improved both my learning efficiency and my understanding of modern cloud architectures.

Personal Experience

The event was highly interactive and featured numerous live demonstrations that made complex technical concepts easier to understand. I especially enjoyed the Advanced Prompt Engineering session because the techniques presented could be immediately applied to everyday work and study.

The CloudFront presentation was equally valuable, providing practical knowledge about content delivery, performance optimization, and security design for modern web applications.

In addition to the technical sessions, the event included a Lucky Draw and networking activities that created a friendly and enjoyable atmosphere. These activities allowed participants to interact with speakers and other community members while sharing learning experiences.

Lessons Learned

  • The effectiveness of AI systems depends heavily on the quality of user prompts.
  • Content Delivery Networks (CDNs) play a critical role in improving application performance and user experience.
  • Security should be incorporated into system architecture from the very beginning of development.
  • Keeping up with emerging technologies is essential for long-term career growth.
  • Hands-on demonstrations provide a much deeper understanding than theoretical study alone.
